## ðŸš€ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/atzin-escandia/atzin-transition-waves.git
cd atzin-transition-waves
```

### 2. Install dependencies

```bash
yarn install
```

### 3. Start the dev server

```bash

```

### 4. Build for production

```bash
yarn build
```

### 5. Launch production server

```bash

```
